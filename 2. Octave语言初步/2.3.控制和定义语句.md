# 控制和定义语句
For i=1:10,  
Indices=a : b 从a到b的索引  
Break Continue 与C语言相同  
While, end 结构体 同C语言  

选择结构：
If condition，  
	command  
	end   

分支选择结构： 
If condition,   
	command;   
elseif condition,   
	Command;  

循环结构：  
While condition，  
	Conmand;  
	 end  

定义函数：  
Function y=function(x)  
	Command with Y,x;  
保存为function.m文件  

返回多个值的函数：  
Function [y1,y2]=function(x)  
	Command with y1,y2,x;  

加载函数： 
定位到m文件目录下  

addpath（'path'） 加入Octave路径
